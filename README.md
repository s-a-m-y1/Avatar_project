# 🎬 Avatar Landing Page

A modern and responsive landing page inspired by the **Avatar** movie theme. This project was built using only **HTML5** and **CSS3** to practice layout design, Flexbox, gradients, hover effects, and responsive styling.

## 🚀 Features

- Clean and modern UI
- Responsive layout
- Navigation bar with hover effects
- Gradient background overlay
- Full-screen hero section
- Stylish call-to-action button
- CSS transitions and hover animations
- Google Fonts integration (Ubuntu)
